# MaMieAdoree Website

Bienvenue sur le site web du projet MaMieAdoree.

## Technologies utilisées

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Installation

1. Clonez le repository :
   ```bash
   git clone https://github.com/adam2504/website_mamieadoree.git
   ```

2. Installez les dépendances :
   ```bash
   npm install
   ```

3. Lancez le serveur de développement :
   ```bash
   npm run dev
   ```

## Déploiement

Le site peut être déployé via des services comme Vercel, Netlify, ou GitHub Pages.
